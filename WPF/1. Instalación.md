# C# - WPF

## 1. Instalación y primeros pasos
1. Instalar las siguientes extensiones:
    - ***.NET Install Tool*:** para la instalación del SDK, [enlace](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.vscode-dotnet-runtime)
    - ***C#*:** para trabajar con C#, [enlace](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp)
2. Una vez instaladas las extensiones, `Ctrl + Shift + P` y seleccionar *.NET Install Tool: Install the .NET SDK System-Wide.* Si no pudieses instalarlo, descárgalo desde este [enlace](https://dotnet.microsoft.com/en-us/download)
3. Seleccionar la última versión
4. Escoger un directorio donde estará la carpeta del proyecto y ejecutar:
    ```bash
    dotnet new wpf -n ProyectoWPF
    cd ProyectoWPF
    ```
    > Cambia *ProyectoWPF* por el nombre que prefieras. En estos ejemplos utilizaré siempre ese nombre
5. Instala la librería de [MaterialDesign](https://github.com/MaterialDesignInXAML/MaterialDesignInXamlToolkit) y luego abre VS Code:
    ```bash
    dotnet add package MaterialDesignThemes
    code .
6. Abrir el archivo *App.xaml* y añadir la línea de MaterialDesign:
    ```xml
    <!-- Observa que se ha añadido xmlns:materialDesign -->
    <Application x:Class="ProyectoWPF.App"
                 xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
                 xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
                 xmlns:materialDesign="http://materialdesigninxaml.net/winfx/xaml/themes"
                 StartupUri="MainWindow.xaml">
        <Application.Resources>
            <!-- Añadir este párrafo -->
            <ResourceDictionary>
                <ResourceDictionary.MergedDictionaries>
                    <materialDesign:BundledTheme BaseTheme="Light" PrimaryColor="Blue" SecondaryColor="Lime" />
                    <ResourceDictionary Source="pack://application:,,,/MaterialDesignThemes.Wpf;component/Themes/MaterialDesign3.Defaults.xaml" /> 
                </ResourceDictionary.MergedDictionaries>
            </ResourceDictionary>

        </Application.Resources>
    </Application>
    ```
7. Abre *MainWindow.xaml* y pon este contenido:
    ```xml
    <Window
        x:Class="ProyectoWPF.MainWindow"
        xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        xmlns:materialDesign="http://materialdesigninxaml.net/winfx/xaml/themes"
        Title="MiProyecto"
        WindowStartupLocation="CenterScreen"
        Height="400"
        Width="600">

        <StackPanel Orientation="Vertical" HorizontalAlignment="Center" VerticalAlignment="Center" >
            <Label Content="Usuario" />
            <TextBox Width="200" />
            <Separator Opacity="0" Height="20"/>
            <Label Content="Contraseña" />
            <PasswordBox />
            <Separator Opacity="0" Height="20"/>
            <CheckBox Content="Recordar sesión" />
            <Separator Opacity="0" Height="20"/>
            <Button Content="Login" />
        </StackPanel>
    </Window>
    ```
8. Ejecutar
    ```
    dotnet run
    ```
9. Por último, trabajaremos con SharpDevelop para el editor XAML. Puedes descargarlo desde este [link](https://sourceforge.net/projects/sharpdevelop/files/latest/download)
